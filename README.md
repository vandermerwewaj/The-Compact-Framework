# The Humanity Forks & Compact Archive
> **"A ground-up redesign of governance as an engineering problem, not a diplomacy problem."**

## 📌 Overview
This repository contains a structured sequence of documents designed to navigate civilizational-scale risks. It moves from a historical analysis of how we arrived at our current volatility to a clinical, adversarial framework for global survival.

The information herein is **non-ideological**. It assumes a future environment of high hostility and low trust. It is not designed to "convert" or "persuade," but to provide a functional, minimum viable architecture for coordination when all other systems fail.

---

## 📂 Document Sequence
To understand the logic of the **Compact**, the documents should be processed in the following order:

1.  **[1. Humanity Forks](./1.%20humanity_forks.md)**
    * *The Historical Divergence:* An analysis of critical decision points in human history, comparing the "Path Taken" vs. the "Optimal Path."
    * **Goal:** To identify the structural patterns of systemic failure.

2.  **[2. Humanity Futures](./2.%20humanity_futures.md)**
    * *The Open Forks:* A real-time tracker of current global trajectories (AI, Bio, Climate) and the narrow windows remaining for intervention.
    * **Goal:** To define the urgency of the current inflection point.

3.  **[3. The Compact Framework](./3.%20the_compact_framework.md)**
    * *The Manifest:* The core structural blueprint for planetary governance. 
    * **Note:** This is the primary document. It is designed to be resilient against regulatory capture and "Great Power" interference. It focuses on **Planetary Commons Risks** only, preserving national sovereignty in all other domains.

4.  **[4. X-Risk Directory](./4.%20xrisk_directory.md)**
    * *Who’s Watching:* A directory of the organizations currently researching, monitoring, and mitigating these risks.
    * **Goal:** Operational resource for finding collaborators and verified data.

## 🛠 Structural Philosophy
This archive was converted from complex interactive HTML to Markdown to ensure maximum accessibility and long-term readability across all platforms. 

* **Verbatim Integrity:** The text of *The Compact* is preserved exactly as engineered. In high-stakes governance, the specific phrasing (the "spells") defines the boundaries that prevent the framework from overreaching into a world government.
* **Adversarial Design:** Unlike traditional treaties, these documents include "Attack Surface" analyses—explicitly detailing how bad actors might try to exploit the system and the mechanical mitigations built in to stop them.
